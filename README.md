# Aryauday Foundation – NGO Website

A modern, responsive single-page website for **Aryauday Foundation**, focused on education, social service, and Gau Raksha.

## Features

- **Blue-themed** design with soft gradients and white sections
- **Fully responsive** (mobile, tablet, desktop)
- **Sticky header** with clean navigation
- **Smooth scrolling** and light animations
- **Sections:** Hero, About, Initiatives, Get Involved, Gallery, Testimonials, Contact, Footer
- **SEO-friendly** meta tags and semantic HTML
- **Fast loading** – minimal dependencies (one font, no heavy JS)

## How to run

1. Open the project folder: `aryauday-foundation`
2. Double-click `index.html` to open in your browser, or use a local server:

   ```bash
   cd aryauday-foundation
   npx serve .
   ```

   Then visit the URL shown (e.g. `http://localhost:3000`).

## Customization

- **Images:** Replace Unsplash URLs in `index.html` with your own image paths.
- **Contact:** Update email, phone, address, and social links in the Contact section and footer.
- **Registration:** Replace the “NGO registration details placeholder” in the footer with your actual registration/80G details.
- **Colors:** Edit CSS variables in `styles.css` (`:root`) to change the blue theme.

## Files

- `index.html` – Main page and content
- `styles.css` – Layout, theme, responsive styles
- `script.js` – Sticky header, mobile menu, form placeholders

No build step required; open `index.html` in a browser to view.
